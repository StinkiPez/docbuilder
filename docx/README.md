# Template Files

Place your Word document template in this folder.

## Required File:
- `Proposal Template.docx` - Your proposal template with docxtemplater tags

The template should contain placeholders like:
- `{client_name}` - Client's full name
- `{project_name}` - Project name
- `{date}` - Current date
- `{project_summary}` - Project summary text
- etc.

For more information on docxtemplater syntax, visit: https://docxtemplater.com/
